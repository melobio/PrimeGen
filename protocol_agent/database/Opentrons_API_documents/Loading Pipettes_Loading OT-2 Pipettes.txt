
This code sample loads a P1000 Single-Channel GEN2 pipette in the left mount and a P300 Single-Channel GEN2 pipette in the right mount. Each pipette uses its own 1000 ÂµL tip rack.

```
from opentrons import protocol_api

metadata = {'apiLevel': '2.16'}

def run(protocol: protocol_api.ProtocolContext):
    tiprack1 = protocol.load_labware(
        load_name='opentrons_96_tiprack_1000ul', location=1)
    tiprack2 = protocol.load_labware(
        load_name='opentrons_96_tiprack_1000ul', location=2)
    left = protocol.load_instrument(
        instrument_name='p1000_single_gen2',
        mount='left',
        tip_racks=[tiprack1])
    right = protocol.load_instrument(
        instrument_name='p300_multi_gen2',
        mount='right',
        tip_racks=[tiprack1])
```
